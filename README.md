🏫 PVPIT Smart Campus Companion
📌 Project Overview
PVPIT Smart Campus Companion is a frontend-based web application designed to help students and visitors easily navigate the campus of Padmabhooshan Vasantdada Patil Institute of Technology (PVPIT), Bavdhan, Pune.
The project focuses on improving campus navigation through a clean, interactive, and responsive user interface.
🎯 Problem Statement
In a large campus environment, new students and visitors often struggle to locate:
Departments
Laboratories
Administrative offices
Common facilities
This project provides a simple and visual solution for easy campus navigation.
💡 Features
🗺️ Campus Map with major blocks
🔍 Smart search for classrooms and labs
🎯 Automatic block highlighting after search
🏢 Clickable blocks with detailed information
📚 Departments section with images
📍 Integrated Google Maps location
🌙 Dark mode toggle
📱 Fully responsive design (Bootstrap)
🛠️ Technologies Used
-HTML5
-CSS3
-JavaScript
-Bootstrap 5
-Google Maps Embed API
